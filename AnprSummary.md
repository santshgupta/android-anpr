# Android ANPR #
Система распознавания номерных знаков

# Информация #
На данный момент реализована нативная работа с графикой:
  * ARMv7 Convolving
  * Thresholding
  * Sobel
  * RGB->HSV convertor
  * RGB565 -> ARGB convertor

Текущая версия без knn-классификатора и наработанной базы для нейросети
